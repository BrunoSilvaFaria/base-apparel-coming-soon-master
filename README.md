## Processo de desenvolvimento:
- Criar a estrutura do site, utilizando HTML;
- Estilizar o site utilizando CSS;
- Verificar o guia de estilo e adaptar;
- Refatorar o CSS e fazer alterações necessárias.

- Arquitetura BEM
